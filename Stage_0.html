Important Concepts

How the Internet Works

The Internet is the interconnection of millions of computers. One giant computer network. Some are consuming content using Web Browsers or Email Clients. Some computers are holding those files, and serving them out when they are needed. These are appropriately called Servers.
When a web browser goes to a url, it is requesting the file(s) for a web site to be sent via HTTP from a server. HTTP is the communication protocal the Internet works on. The server finds the need file(s), then send them back so the web browser can interpet them so they make sense to us lowly humans.

HTML

HTML stands for Hypertext Markup Language. HTML pages form the vast majority of all content on the Internet.

Tags and Elements

The parts of an HTML pages, other than the text, is called Elements. This is the combination of Tags, and other things like values and URL's. Tags are used to tell the web browser the type of Element that is being seen.

Inline and Block Elements

Elements in HTML are either Inline or Block. Inline elements are used inside of the text itself. Block elements form a box or block around the content inside of them.
